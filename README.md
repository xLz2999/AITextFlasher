# AITextFlasher (AI文本转换器)⚡

✨ **为 AI 重度使用者打造的轻量级 Markdown 转换神器** ✨

## 🌟 为什么做这个项目？
作为一个频繁与 AI 对话的用户，我受够了每次要把 AI 的回答（Markdown 格式）复制到复杂的编辑器里，然后再转成 Word 的繁琐流程。
为了“吃自己的狗粮（Dogfooding）”，我亲自上手开发了这个小工具！这是我的**第一个开源项目**，主打一个“轻量、极速、无广告”。

## 🚀 核心功能
- **极速纯文本提取**：一键剥离所有 Markdown 符号（如 `**`、`#`），直接将最纯净的文本送入系统剪贴板。
- **一键无损导 Word**：保留原有的标题、加粗、列表排版，瞬间生成电脑可读的 `.docx` 文件。
- **原生极速体验**：启动速度秒杀大型文本编辑器，用完即走，绝不拖泥带水！

## 🛠️ 技术栈
- **C# / .NET** (底层核心逻辑)
- **Avalonia UI** (现代化跨平台桌面界面框架)
- **Markdig & HtmlToOpenXml** (文档解析与生成核心引擎)
