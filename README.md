# AITextFlasher (AI文本转换器)

✨ **为 AI 使用者打造的轻量级 Markdown 转换器** ✨

## 🚀 核心功能
- **纯文本提取**：一键剥离所有 Markdown 符号（如 `**`、`#`），将文本送入系统剪贴板。
- **导 Word**：保留原有的标题、加粗、列表排版，瞬间生成电脑可读的 `.docx` 文件。

## 🛠️ 技术栈
- **C# / .NET** (底层核心逻辑)
- **Avalonia UI** (现代化跨平台桌面界面框架)
- **Markdig & HtmlToOpenXml** (文档解析与生成核心引擎)
