# AITextFlasher (AI文本转换器)

✨ **为 AI 使用者打造的轻量级 Markdown 转换器** ✨

## 🚀 功能
- **文本提取**：剥离所有 Markdown 符号（如 `**`、`#`），将文本送入系统剪贴板。
- **导出 Word**：保留原有的标题、加粗、列表排版，生成 `.docx` 文件。

## 🛠️ 技术栈
- **C# / .NET** (底层核心逻辑)
- **Avalonia UI** (跨平台桌面界面框架)
- **Markdig & HtmlToOpenXml** (文档解析与生成引擎)
